# Running time order
| Selection Sort | Bubble Sort | Merge Sort |
| :------------: | :---------: | :--------: |
|     O(n²)      |    O(n²)    | O(n log n) |
|     Ω(n²)      |    Ω(n)     | Ω(n log n) |
